Minecraft YouTuber LEGO Set

I designed a custom LEGO set inspired by popular Minecraft Let's Play YouTubers. The featured location is “L'Manberg,” a key setting from the Dream SMP server, recreated to reflect scenes from their videos.

Details

Minifigures

Each character was individually modeled to reflect their Minecraft persona, using Blender and Photoshop.

Project Highlights

The set gained significant attention on Twitter (now X), reaching the trending tab with over 30,000 likes and 1,000+ shares. Even Tommyinnit himself Replied wanting the set to be made officially.
